Employee Leave & Meal Subscription Management System
📌 Overview

This project is a Spring Boot-based backend application that manages employee operations along with an integrated meal subscription system.

It allows employees to:

Manage their profiles
Apply for leaves
Subscribe to meals
🚀 Features
👨‍💼 Employee Management
Add, update, and delete employees
View employee details
Secure authentication (JWT / Basic Auth)
🗓️ Leave Management
Apply for leave
Approve/reject leave requests
Track leave history
🍱 Meal Subscription
Subscribe to meals
Manage daily/weekly meal plans
Link meals with employees
